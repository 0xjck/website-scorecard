Capture Detailed Website Screenshots

This tool will create high-quality screenshots of the detailed websites for your scorecard.

Instructions:

  1. Click "Generate Old Website Screenshot" to create old-website.png
  2. Click "Generate New Website Screenshot" to create new-website.png
  3. Both files will be automatically downloaded
  4. Move the files to your website-scorecard folder if needed
  5. Refresh the scorecard to see the results
